### 8. Property Evaluation

**Location:** `Property Evaluation/`

Tools for analyzing deals and determining profitability.

| Document | Purpose |
|----------|---------|
| **PRES (Property Repair Estimate Sheet).pdf** | Detailed property inspection form covering roof, exterior, HVAC, plumbing, electrical, interior, and more - calculates total repair costs |
| **Copy of Fix and Flip Rules.docx** | Investment guidelines and rules-of-thumb (70% rule, profit margins, etc.) |
| **Copy of Copy of Flip Analysis.xlsx** | Spreadsheet for calculating flip profitability (purchase → repairs → ARV → profit) |
| **Austin Property Evaluation.xlsx** | Market-specific property analysis for Austin area deals |

**Evaluation Workflow:**
1. Use **PRES sheet** to inspect property and estimate repairs
2. Apply **Fix and Flip Rules** to check if deal meets criteria
3. Run numbers in **Flip Analysis** spreadsheet
4. Reference **Austin Property Evaluation** for market comparisons
