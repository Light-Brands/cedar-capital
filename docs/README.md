## Deal Workflow Overview

```
1. LEAD GEN          Call sellers using Sales Playbook
        ↓
2. ANALYZE           Evaluate property using Deal Analysis guide
        ↓
3. CONTRACT          Execute Texas SFR Purchase Contract
        ↓
4. REHAB/FLIP        Budget repairs, track expenses
        ↓
5. DISPOSITION       Match with buyers, close deal
```
