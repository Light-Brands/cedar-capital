# Cedar Capital Deal Analysis Guide

## The 70% Rule

**Maximum Offer = (ARV × 70%) - Repairs**

| ARV | Max Offer (before repairs) |
|-----|---------------------------|
| $200,000 | $140,000 |
| $300,000 | $210,000 |
| $400,000 | $280,000 |
| $500,000 | $350,000 |

*Example: ARV $300K, Repairs $40K → Max Offer = $210K - $40K = **$170K***

## Deal Criteria

### Must-Have
- [ ] ARV verified with 3+ comps
- [ ] 70% rule works (or better)
- [ ] Clear title (or path to clear)
- [ ] No major structural issues
- [ ] Seller motivated

### Red Flags
- Foundation problems requiring specialist
- Environmental issues (mold, asbestos, lead)
- Title clouds without clear resolution
- Seller unrealistic on price
- Property in declining area
