# Cedar Capital Website

A Next.js website for Cedar Capital - local Austin home buyers offering fast, fair cash offers.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS with custom brand design system
- **Animations**: Framer Motion
- **TypeScript**: Full type safety

## Brand Design System

The design system follows Cedar Capital's brand guidelines:

### Colors
- **Primary**: Cedar Green (#0B3D2E), Capital Gold (#C8A24A)
- **Secondary**: Evergreen (#1F5B47), Navy Ink (#0F2233)
- **Neutral**: Cream (#F6F1E6), Sand (#E9DDC7), Stone (#D2C7B3), Charcoal (#1F2933)

### Typography
- **Headlines**: Plus Jakarta Sans (600-700 weight)
- **Body**: Inter (400-500 weight)

### Design Principles
- Trustworthy like a bank
- Warm like a local family
- Modern + streamlined

## Pages

- `/` - Homepage with hero, value propositions, and audience split
- `/about` - Company story, values, and mission
- `/sellers` - Information and lead capture for homeowners
- `/investors` - Deal flow and investor network signup
- `/contact` - Contact form and FAQ

## Development

```bash
# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Run linting
npm run lint

# Type check
npm run type-check
```
